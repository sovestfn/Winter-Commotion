# WinterCommotion
Проект содержит в себе основной и обучающий уровни гиперказуальной игры "Winter Commotion" 
1. Основное меню:
	Начать игру
	Обучающий уровень
	Настройка уровня сложности игры
	Авторы
	Выход
1.1. Меню паузы:
	Продолжить игру
	Выйти в главное меню
	Выход из игры
2. Игровые механики (некоторые есть только на обучающем уровне, 
т.к. предполагается их использование в последующих уровнях):
	Сбор предметов и увеличение счета(подарки)
	Уворачивание от летающих снарядов(снежки)
	Объекты изменяющие скорость игрока (лед - ускорение, сугроб - замедление)
3. Звуки:
	Музыка в главном меню
	Музыка на игровом и обучающем уровнях
	Звуки взаимодействия с меню
	Звук взаимодействия с предметом (подарок)
	Звук получения урона (снежок)
	Звук соприкосновения снежка с объектом, отличающимся от игрока
	Звук поражения
	Шаги
4. Визуальные эффекты:
	Взаимодействие с предметом (подарок)
	Получение урона от снежка
	Смерть персонажа
